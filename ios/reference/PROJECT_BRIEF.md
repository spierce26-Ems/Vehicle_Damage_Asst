# Vehicle Damage Forensic Matching System

## Project Origin
User's vehicle was hit while parked. Need forensic-level matching system to link damage patterns between victim and suspect vehicles.

## Core Concept
Like ballistic matching (bullet → gun), match paint transfer, impact geometry, height alignment, and deformation patterns between two vehicles with high confidence scoring.

## Technology Stack
- iPhone sensors: LiDAR, high-res cameras, gyroscope, accelerometer, depth sensing
- Real-time guidance for optimal photo angles
- Computer vision + 3D reconstruction
- Forensic analysis algorithms

## Target Markets
1. Insurance adjusters
2. Law enforcement agencies  
3. Private investigators
4. Body shops / collision centers
5. Legal evidence documentation

## Key Innovation
Real-time sensor fusion guides user to capture forensically-valid evidence with proper angles, lighting, and measurements.
